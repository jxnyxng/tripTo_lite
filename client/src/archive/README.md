# Archive 폴더

이 폴더는 메인 개발에서 사용되지 않는 파일들을 정리한 아카이브입니다.

## 폴더 구조

### `/versions/`
- 이전 버전의 ResultPage 파일들
- `ResultPage_old.js` - 오래된 백업 버전
- `ResultPage_withBoxes.js` - 외부 박스가 있던 이전 버전

### `/experimental/`
- 실험적이거나 테스트용 파일들
- `ResultPage_experimental.js` - 실험적 구현 버전

## 현재 사용 중인 파일

메인 src 폴더의 `ResultPage.js`가 현재 활성 파일입니다.
